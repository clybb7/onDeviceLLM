A local LLM named Steven that runs entirely offline, giving users access to an AI assistant while maintaining complete privacy for the user. 

This is a wrapper for Llama and Deepseek LLMs, allowing for general chatting, simple internet searches and simple code writing/debugging. Requires 16+ GB RAM. 

Requires Python 3.12 or earlier. 

You will need the following models downloaded on your machine: 

https://huggingface.co/bartowski/Meta-Llama-3.1-8B-Instruct-GGUF

https://huggingface.co/TheBloke/deepseek-coder-6.7B-instruct-GGUF
